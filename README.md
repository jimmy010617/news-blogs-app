## 메인 페이지
> 메인 프레임워크 : React + Vite

> 페이지 디자인 : CSS

> 포스팅 작성, 수정, 저장 기능 : JavaScript

## 사용 API
> 뉴스 API : GNews

> 날씨 API : openweathermap

### 실행방법
> cd news-blogs-app

> npm run dev
